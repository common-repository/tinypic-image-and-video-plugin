This plugin integrates the Tinypic.com plugin into WordPress. Video and Image
uploads are supported. 

== Description ==

This plugin is designed to allow content publishers using wordpress as their publishing tool the ability to seamlessly use 
tinypic.com for all of their image and video hosting needs.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `tinypic.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
